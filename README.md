# Learn How to Program 

First week of rails project to create a clone of the Epicodus site www.learnhowtoprogram.com. 

## Usage

To run this project you must have rails installed.

Open project in terminal and run bundle to install necessary gems and run the following in terminal.

```
rake db:create
rake db:seed
rake db:migrate
rails s
```

You should then be able to open on localhost:3000. You can then add lessons and tasks for each lesson.
